# Система анализа энергопотребления квартиры

## Описание

Веб-приложение для мониторинга, анализа и оптимизации энергопотребления в квартире. Позволяет отслеживать потребление электроэнергии различными приборами, рассчитывать стоимость с учётом многозонных тарифов, получать рекомендации по энергосбережению.

## Основные возможности

- **Аутентификация**: регистрация и авторизация пользователей
- **Управление приборами**: добавление, редактирование, удаление электроприборов
- **Группировка**: организация приборов по комнатам/категориям
- **Многозонные тарифы**: настройка пиковых, ночных и полупиковых тарифов
- **Аналитика**: графики потребления по дням и месяцам
- **Прогнозирование**: расчёт ожидаемого потребления и стоимости
- **Рекомендации**: персонализированные советы по энергосбережению
- **Экспорт**: выгрузка данных в CSV формат

## Технологии

- **Backend**: Java Servlets, JDBC
- **Frontend**: HTML5, CSS3, JavaScript (без фреймворков)
- **База данных**: MySQL 8.0+
- **Сервер**: Apache Tomcat 9.0+

## Быстрый старт

1. Установите JDK 11+, MySQL 8.0+, Apache Tomcat 9.0+
2. Создайте базу данных и выполните `database/schema.sql`
3. Настройте подключение в `DatabaseConnection.java`
4. Скомпилируйте и разверните WAR-файл на Tomcat
5. Откройте http://localhost:8080/energy-analysis/

Подробная инструкция: [docs/INSTALLATION.md](docs/INSTALLATION.md)
